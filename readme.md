# Hildegard Orgon Akkumulator Plugin

# Filter

### Bit Crusher (bit_crusher)

#### Params

- factor ```int``` (z.B.: 2, 4, 8, 16, etc...) (0 = off)

### Bitrate (bitrate)

#### Params

- quality ```float``` 0 - 1

### DeepFry (deepfry)

#### Params

- drive ```float``` (pre-gain faktor)
- mix ```float``` (blend faktor zwischen original und distorted signal)

### Lua (lua)

> EXPERIMENTAL! Leute ***können*** damit Blödsinn machen  
> (Ist aber eine Sandbox, also kein Zugriff nach außen)

#### Params

- code ```string```
